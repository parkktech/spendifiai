  <action>
    **AccountingFirm model** (`app/Models/AccountingFirm.php`):
    - Fillable: name, address, phone, logo_url, primary_color, invite_token
    - Auto-generate invite_token via `Str::random(64)` in `booted()` creating callback
    - Relationships: `members()` hasMany User (accounting_firm_id), `documentRequests()` hasMany DocumentRequest
    - $hidden: invite_token (only exposed via dedicated endpoint)

    **DocumentAnnotation model** (`app/Models/DocumentAnnotation.php`):
    - Fillable: tax_document_id, user_id, parent_id, body
    - Relationships: `document()` belongsTo TaxDocument, `author()` belongsTo User, `parent()` belongsTo self, `replies()` hasMany self ordered by created_at
    - Eager load author on queries

    **DocumentRequest model** (`app/Models/DocumentRequest.php`):
    - Fillable: accounting_firm_id, client_id, accountant_id, description, tax_year, category, status, fulfilled_document_id
    - Cast status to DocumentRequestStatus enum
    - Relationships: `firm()` belongsTo AccountingFirm, `client()` belongsTo User, `accountant()` belongsTo User, `fulfilledDocument()` belongsTo TaxDocument (nullable)
    - Scope: `scopePending($query)` where status = pending

    **DocumentRequestStatus enum** (`app/Enums/DocumentRequestStatus.php`):
    - PHP 8.3 backed string enum with cases: Pending = 'pending', Uploaded = 'uploaded', Dismissed = 'dismissed'
    - Add `label()` method returning human-readable labels

    **User model changes** (`app/Models/User.php`):
    - Add `accounting_firm_id` to $fillable
    - Add `accountingFirm()` belongsTo AccountingFirm relationship
    - Add `documentAnnotations()` hasMany DocumentAnnotation relationship
    - Add `documentRequests()` hasMany DocumentRequest (as client)

    **Migrations:**
    1. `create_accounting_firms_table`: id, name (string 255), address (text nullable), phone (string 20 nullable), logo_url (string 500 nullable), primary_color (string 7 nullable), invite_token (string 64 unique), timestamps
    2. `add_accounting_firm_id_to_users`: Add accounting_firm_id (unsignedBigInteger nullable, FK -> accounting_firms, onDelete set null)
    3. `create_document_annotations_table`: id, tax_document_id (FK -> tax_documents onDelete cascade), user_id (FK -> users onDelete cascade), parent_id (FK -> document_annotations nullable onDelete cascade), body (text), timestamps. Indexes: (tax_document_id, created_at), (parent_id)
    4. `create_document_requests_table`: id, accounting_firm_id (FK -> accounting_firms), client_id (FK -> users), accountant_id (FK -> users), description (text), tax_year (integer nullable), category (string nullable), status (string 20 default 'pending'), fulfilled_document_id (FK -> tax_documents nullable), timestamps. Indexes: (client_id, status), (accounting_firm_id)

    Run `php artisan migrate` after creating all migrations.
