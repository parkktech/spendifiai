**4. Create ExtractTaxDocument job** at `app/Jobs/ExtractTaxDocument.php`:
- Implements ShouldQueue, uses Dispatchable, InteractsWithQueue, Queueable, SerializesModels
- `$tries = 3`, `$timeout = 120`, `$backoff = [10, 30, 60]` (exponential)
- Constructor takes `int $documentId`
- `handle(TaxDocumentExtractorService $extractor, TaxVaultAuditService $audit)`:
  1. Load document via TaxDocument::findOrFail
  2. Set status to Classifying, call $extractor->classify()
  3. Update category and classification_confidence
  4. If confidence < config('spendifiai.ai.extraction_thresholds.classification_gate', 0.70), set status to Failed, return
  5. Set status to Extracting, call $extractor->extract()
  6. Update extracted_data and status to Ready
  7. Audit log: action 'extraction_completed' with category and overall_confidence
- `failed(\Throwable $exception)`: Set document status to Failed, log error

  ```php
  public function updateField(UpdateExtractionFieldRequest $request, TaxDocument $document): JsonResponse
  {
      $this->authorize('view', $document);
      $validated = $request->validated();
      $data = $document->extracted_data ?? ['fields' => []];
      $oldValue = $data['fields'][$validated['field']]['value'] ?? null;
      $data['fields'][$validated['field']] = [
          'value' => $validated['value'],
          'confidence' => 1.0,
          'verified' => true,
      ];
      $document->update(['extracted_data' => $data]);
      $this->auditService->log($document, $request->user(), 'field_corrected', $request, [
          'field' => $validated['field'],
          'old_value' => $oldValue,
          'new_value' => $validated['value'],
      ]);
      return response()->json(['success' => true]);
  }
  ```
- Add `acceptAll()` method:
  ```php
  public function acceptAll(Request $request, TaxDocument $document): JsonResponse
  {
      $this->authorize('view', $document);
      $data = $document->extracted_data ?? ['fields' => []];
      foreach ($data['fields'] as $fieldName => &$field) {
          if (!($field['verified'] ?? false)) {

      }
      $document->update(['extracted_data' => $data]);
      $this->auditService->log($document, $request->user(), 'fields_accepted', $request);
      return response()->json(['success' => true]);
  }
  ```

```php
public function retryExtraction(Request $request, TaxDocument $document): JsonResponse
{
    $this->authorize('view', $document);
    if ($document->status !== DocumentStatus::Failed) {
        return response()->json(['error' => 'Document is not in failed state'], 422);
    }
    $document->update(['status' => DocumentStatus::Upload->value]);
    ExtractTaxDocument::dispatch($document->id);
    return response()->json(['success' => true, 'message' => 'Extraction re-queued']);
}
```
