    **TaxVaultStorageService** (`app/Services/TaxVaultStorageService.php`):
    Handles file storage abstraction (local vs S3) with signed URL generation.

    Methods:
    - `store(UploadedFile $file, int $userId, int $taxYear, ?string $category = null): array` -- Returns `['stored_path' => string, 'disk' => string, 'file_hash' => string, 'file_size' => int, 'mime_type' => string]`. Path format: `tax-vault/{userId}/{taxYear}/{category ?? 'uncategorized'}/{uuid}.{ext}`. Computes sha256 hash of file contents. Uses `Storage::disk($this->getActiveDisk())->putFileAs()`.
    - `getSignedUrl(TaxDocument $document): string` -- For local disk: use `URL::temporarySignedRoute('tax-vault.download', now()->addMinutes(config('spendifiai.vault.signed_url_expiry_minutes')), ['document' => $document->id])`. For S3 disk: use `Storage::disk('s3')->temporaryUrl($document->stored_path, now()->addMinutes(config('spendifiai.vault.signed_url_expiry_minutes')))`.
    - `delete(TaxDocument $document): void` -- Deletes file from the active disk.
    - `getActiveDisk(): string` -- Returns `config('spendifiai.vault.storage_driver')` (either 'local' or 's3').
    - `migrateDocument(TaxDocument $document, string $targetDisk): void` -- Copies file from current disk to target disk, updates document's `disk` and `stored_path`, deletes from source. Used by the admin migration job.
    - `testS3Connection(): bool` -- Attempts to put/get/delete a test file on S3 disk. Returns true on success, throws exception with message on failure.
    - `getStorageStats(): array` -- Returns `['total_documents' => int, 'total_size_bytes' => int, 'active_driver' => string]` from database aggregation.

    Private:
    - `getDisk(string $driver): \Illuminate\Contracts\Filesystem\Filesystem` -- Returns Storage::disk() for the given driver. For local, uses 'local' disk with the vault path prefix. For S3, uses the 's3' disk.
    - `validateFile(UploadedFile $file): void` -- Validates MIME type against config('spendifiai.vault.allowed_mimes') and size against config('spendifiai.vault.max_file_size_mb'). Throws ValidationException on failure.

    **TaxVaultAuditService** (`app/Services/TaxVaultAuditService.php`):
    Handles audit logging with hash chain integrity.

    Methods:
    - `log(TaxDocument $document, User $user, string $action, ?Request $request = null, array $metadata = []): TaxVaultAuditLog` -- Creates an immutable audit entry. Steps: (1) Get the last audit entry for this document to retrieve previous_hash, (2) Compute entry_hash as `hash('sha256', ($previousHash ?? 'genesis') . '|' . $document->id . '|' . $user->id . '|' . $action . '|' . now()->toISOString())`, (3) Create the TaxVaultAuditLog record with all fields. Extract IP from `$request->ip()` and user agent from `$request->userAgent()` if request provided.
    - `verifyChain(TaxDocument $document): array` -- Returns `['valid' => bool, 'entries_checked' => int, 'first_broken_at' => ?int]`. Iterates all audit entries for the document in created_at order, recomputes each hash, compares to stored hash. First entry should use 'genesis' as previous hash.
    - `getLogForDocument(TaxDocument $document, ?User $viewer = null): Collection` -- Returns audit entries. If viewer is Super Admin (user_type === 'admin'), make ip_address and user_agent visible. Otherwise return with those fields hidden (default $hidden behavior).
