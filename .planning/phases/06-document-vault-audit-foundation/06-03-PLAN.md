    - Outer: `bg-sw-card rounded-lg border border-sw-border p-4 cursor-pointer hover:border-sw-accent transition`

    Wraps FileDropZone for vault context. Props: `taxYear: number`, `onUploadComplete: () => void`. Renders FileDropZone with `maxSizeMb={100}`, `multiple={true}`, `acceptedExtensions={['.pdf', '.jpg', '.jpeg', '.png']}`, `acceptedMimes={['application/pdf', 'image/jpeg', 'image/png']}`. On file selection, POSTs to `/api/v1/tax-vault/documents` with FormData (file + tax_year). Shows per-file upload progress bar using axios `onUploadProgress`. After all uploads complete, calls `onUploadComplete` to refresh document list. Error handling: show upload errors per-file with red text.

    **MissingAlertBanner** (`MissingAlertBanner.tsx`):
    Persistent top banner. Props: `alerts: { message: string; details?: string }[]`. Renders as a yellow/amber alert bar above the grid: `bg-amber-50 border border-amber-200 rounded-lg p-3`. Shows count ("2 expected documents missing for 2025") with expandable details section. Uses AlertTriangle icon from lucide-react. If no alerts, renders null.

    Table component for audit trail. Props: `entries: TaxVaultAuditEntry[]`, `isLoading?: boolean`. Columns: Action (with human-readable label), User Name, Date/Time (formatted). Rows sorted by created_at descending. Use existing table styling pattern from the app. Loading state shows skeleton. Empty state shows "No audit entries yet."
